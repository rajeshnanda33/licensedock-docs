# Plans & Pricing

Each product has one or more plans. Each plan has one or more prices.

## Plans

A plan represents a tier or variant of your product.

### Activation Type

Each plan has an **activation type** that controls how you want to track usage. For example, a Joomla extension is typically tracked by domain (one site = one activation), while a desktop app is tracked by device.

| Type | What it tracks | Example |
|------|---------------|---------|
| Domain | Websites | `example.com` |
| Device | Computers or phones | `MacBook-Pro-ABC123` |
| Seat | Individual users | `user@company.com` |
| Instance | Servers | `server-prod-01` |

### Activation Limit

The maximum number of activations per license. Set to `0` for unlimited.

## Prices

Each plan can have multiple prices with different billing cycles.

### Billing Cycles

| Cycle | Description |
|-------|-------------|
| One-time | Customer pays once and gets lifetime access |
| Monthly | Billed every month |
| Quarterly | Billed every 3 months |
| Semi-annual | Billed every 6 months |
| Annual | Billed every year |

The billing cycle determines whether a customer pays once or on a recurring schedule.

### Setting Up Prices

1. Open a plan
2. Click **Add Price**
3. Set the amount and billing cycle
4. Save

You can offer multiple billing cycles on the same plan – for example, both monthly and annual pricing. Customers choose at checkout.

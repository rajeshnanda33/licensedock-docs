# Configuration

Open the module settings at **Content > Site Modules > Anything Slider**.

## Slider Settings

| Setting | Description | Default |
|---------|-------------|---------|
| Auto Slide | Automatically advance slides | Yes |
| Slide Delay | Time between transitions (seconds) | 5 |
| Pause on Hover | Pause auto-slide when the mouse hovers over the slider | Yes |
| Continuous Loop | Loop back to the first slide after the last | Yes |
| Random Order | Shuffle slide order on each page load | No |
| Start Random | Begin on a random slide instead of the first | No |

## Transition Settings

| Setting | Description | Default |
|---------|-------------|---------|
| Fade Speed | Duration of the fade transition (milliseconds) | 500 |

## Navigation

| Setting | Description | Default |
|---------|-------------|---------|
| Navigation Arrows | Show, hide, or show on hover | Show on Hover |
| Navigation Dots | Show clickable dots for each slide | Yes |
| Dots Position | Where to place the dots – 6 positions available (top-left, top-centre, top-right, bottom-left, bottom-centre, bottom-right) | Bottom Centre |

## Appearance

| Setting | Description | Default |
|---------|-------------|---------|
| Slider Height | Height of the slider container | 400px |
| Show Loading Icon | Display a loading spinner during auto-slide | Yes |
| Custom CSS | Add your own CSS rules to style the slider | – |

## Adding Slides

Each slide supports:

- **Background Image** – the main slide image
- **Overlay Elements** – text, buttons, or images positioned on top with animation

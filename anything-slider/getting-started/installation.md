# Installation

## Requirements

- Joomla 5.x
- PHP 8.1 or higher

## Install via Joomla Installer

1. Download the `mod_anything_slider` package from [dotscastle.com](https://dotscastle.com)
2. In Joomla admin, go to **System > Install > Extensions**
3. Upload the package file and click **Upload & Install**
4. The module is now available under **Content > Site Modules**

## Publish the Module

1. Go to **Content > Site Modules**
2. Find **Anything Slider** in the list
3. Click it to open the module settings
4. Set **Status** to **Published**
5. Assign it to a **Position** in your template
6. Choose which **Menu Items** should display the slider
7. Click **Save**

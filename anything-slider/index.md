# Anything Slider

Anything Slider is a Joomla 5 module for creating image sliders with animated elements, smooth transitions, and flexible navigation controls – no coding required.

## Features

- **Auto-Slide** – Automatic slide transitions with configurable delay, pause on hover, continuous looping, and random start options.
- **Navigation Controls** – Show or hide navigation arrows and dots. Position dots in 6 locations – top/bottom corners or centre.
- **Smooth Transitions** – Fade transitions with customisable speed control for a polished, professional look.
- **Custom Styling** – Set container height, add custom CSS, and style the slider to match your site design.

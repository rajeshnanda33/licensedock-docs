# Quick Start

Get your first product live in 5 minutes.

## 1. Create a Product

1. Go to **Components → LicenseDock → Products**
2. Click **New**
3. Fill in the product name and description
4. Save the product

## 2. Add a Plan

1. On the product page, click **Add Plan**
2. Set the plan name (e.g., "Single Site", "Unlimited")
3. Choose the **Activation Type** – how you want to track usage (by domain, device, user, or server instance)
4. Set the activation limit (e.g., 1 for single site, 0 for unlimited)
5. Save the plan

## 3. Add a Price

1. On the plan page, click **Add Price**
2. Set the amount
3. Choose the **Billing Cycle** – one-time, monthly, quarterly, semi-annual, or annual
4. Save the price

## 4. Upload a Download

1. Go to the product's **Downloads** tab
2. Upload your file (ZIP, etc.)
3. Set the version number
4. Save

## 5. Create a Menu Item

1. Go to **Menus → Main Menu → Add New Menu Item**
2. Select **LicenseDock → Product** as the menu item type
3. Choose your product
4. Save

## 6. Test a Purchase

1. Visit the product page on your site
2. Click **Buy Now**
3. Complete the checkout using your test gateway credentials
4. Verify the order appears in **Components → LicenseDock → Orders**

You're live! Check the rest of the docs for advanced configuration.
